La macchina tipo "Carolina" si scassa se accelleri troppo.
Quando l'accelerazione è > MAX_SPEED si scassa e si resetta a zero. In qualità di gino il Pilotino, fai tutto il necessario per mandarla oltre iil suo limte. Qualsiasi cosa nel codice è ammessa ( salvo cambiare i sorgenti della Carolina ). Qualsiasi trucco fuori dal codice no.

Riesci a farla andare oltre il suo limite ? L'obiettivo è di avere scritto "Boost!" sulla console.

Operate: to compile

  1. maven

    mvn clean compile

  2. javac

    mkdir -p target/classes
    javac -d target/classes -sourcepath src/main/java/ src/main/java/com/mondora/quiz/*.java


to run
  1. maven

    mvn clean compile exec:java

  2. javac

    java -Djava.security.manager -cp target/classes/ com.mondora.quiz.Pilotino
